# Bitnami Secure Image for Haskell Dockerfile Linter

## What is Haskell Dockerfile Linter?

> Hadolint is an open-source Dockerfile linter that checks best practices, flags common mistakes, and uses ShellCheck to validate shell commands inside RUN instructions.

This container image is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
