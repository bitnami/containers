# Bitnami Secure Image for Dagger

## What is Dagger?

> Dagger is a programmable automation engine for building, testing, and shipping software using containers, with caching and observability across local and CI environments.

This container image is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
