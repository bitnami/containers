# Bitnami Secure Image for Elastic Elasticsearch

## What is Elastic Elasticsearch?

> Elasticsearch is a distributed search and analytics engine part of the Elastic Stack. It is used for web search, log monitoring, and real-time analytics. Ideal for Big Data applications.

[Overview of Elastic Elasticsearch](https://www.elastic.co/elasticsearch)
