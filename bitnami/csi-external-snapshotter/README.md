# Bitnami Secure Image for CSI External Snapshotter

## What is CSI External Snapshotter?

> CSI External Snapshotter is a sidecar that manages volume snapshots for CSI drivers via VolumeSnapshot CRDs and CSI snapshot operations.

This container image is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
