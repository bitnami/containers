# Bitnami Secure Image for Crossplane

## What is Crossplane?

> Crossplane is a framework for building without needing to write code cloud native control planes that can orchestrate applications and infrastructure no matter where they run.

This container image is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
