# Bitnami Secure Image for Elastic Kibana

## What is Elastic Kibana?

> Kibana is an open source, browser based analytics and search dashboard for Elastic Stack. Kibana strives to be easy to get started with, while also being flexible and powerful.

[Overview of Elastic Kibana](https://www.elastic.co/kibana)
