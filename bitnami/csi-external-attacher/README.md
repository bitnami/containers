# Bitnami Secure Image for CSI External Attacher

## What is CSI External Attacher?

> CSI External Attacher is a sidecar that watches VolumeAttachment objects and triggers ControllerPublish/Unpublish against a CSI endpoint.

This container image is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
