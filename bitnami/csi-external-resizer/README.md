# Bitnami Secure Image for CSI External Resizer

## What is CSI External Resizer?

> CSI External Resizer is a sidecar that watches PersistentVolumeClaims and triggers controller-side volume expansion against a CSI endpoint.

This container image is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
