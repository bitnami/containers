# Bitnami Secure Image for Kyverno Chainsaw

## What is Kyverno Chainsaw?

> Kyverno Chainsaw is a declarative, low-code tool for end-to-end testing of Kubernetes controllers and operators. It uses YAML to define test steps, assertions, and resource states without requiring external programming languages.

This container image is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
