# Bitnami Secure Image for Eclipse Temurin minimal

## What is Eclipse Temurin minimal?

> Eclipse Temurin is a high-performance Java runtime (JDK/JRE) built from OpenJDK, offering enterprise-grade stability and security. This ultra-lightweight option is ideal for security-focused and resource-constrained environments.

This container image is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
