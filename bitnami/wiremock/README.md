# Bitnami Secure Image for WireMock

## What is WireMock?

> WireMock is an open-source tool for simulating HTTP-based APIs. It lets you mock external services to isolate your app during development and testing, featuring request matching, response stubbing, record/playback, and fault injection.

This container image is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
