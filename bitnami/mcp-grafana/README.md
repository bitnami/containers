# Bitnami Secure Image for Grafana MCP Server  

## What is Grafana MCP Server?

> Grafana MCP Server is a Model Context Protocol server that lets AI tools query, manage, and interact with Grafana dashboards, datasources, alerts, logs, and incidents


This container image is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
