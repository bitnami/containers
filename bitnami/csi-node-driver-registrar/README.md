# Bitnami Secure Image for CSI Node Driver Registrar

## What is CSI Node Driver Registrar?

> CSI Node Driver Registrar is a sidecar that registers a CSI driver with the kubelet using the kubelet plugin registration mechanism.

This container image is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
